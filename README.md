## This is my personal website: [apollointhehouse.me](apollointhehouse.me)
Made using the  [Kobweb](https://github.com/varabyte/kobweb) framework.

## Getting Started
First,  make sure you have installed the kobweb [binary](https://github.com/varabyte/kobweb#:~:text=Install%20the%20Kobweb%20binary).

Then, run the development server by typing the following command in a terminal under the `site` folder:

```bash
$ cd site
$ kobweb run
```

Open [http://localhost:8080](http://localhost:8080) with your browser to see the result.

Press `Q` in the terminal to gracefully stop the server.
